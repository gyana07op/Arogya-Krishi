# 🌱 Arogya Krishi

### AI-Powered Plant Disease Diagnosis Assistant 🚜

Arogya Krishi is an AI-powered agricultural assistant designed to help farmers, gardeners, and plant owners identify common plant diseases through symptom-based diagnosis and receive practical treatment recommendations.

🔗 Live Demo: https://arogya-krishi.onrender.com/

---

## ✨ Features

✅ AI-powered conversational chatbot

✅ Detects common plant diseases

✅ Symptom-based diagnosis system

✅ Farmer-friendly treatment suggestions

✅ Prevention and crop care tips

✅ Supports multiple crops and plants

✅ Simple and interactive UI

---

## 🌿 Supported Diseases

The system currently supports diagnosis for:

* Powdery Mildew
* Root Rot
* Leaf Spot Disease
* Nitrogen Deficiency
* Underwatering
* Aphid Infestation
* Early Blight
* Downy Mildew
* Bacterial Wilt
* General Pest Damage

## 🧠 How It Works

Arogya Krishi uses:

* AI-powered conversational interaction
* Rule-based expert system logic
* Symptom analysis
* Prompt engineering techniques

The chatbot asks users about:

* plant type
* visible symptoms
* soil condition
* leaf appearance

Then it predicts the most likely disease and suggests treatments and prevention methods.
